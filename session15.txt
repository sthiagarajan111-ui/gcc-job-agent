═══════════════════════════════════════════════════════
FUNCTIONS TO EXPORT
═══════════════════════════════════════════════════════

1. async generateCoverLetter(job)
   Input: job object with fields:
     title, company, location, description, tier, tierLabel
   Calls Claude API (claude-sonnet-4-20250514) with:
     - system prompt containing Dheeraj's full profile
       and cover letter writing instructions above
     - user message: "Write a cover letter for this job:
       Title: [title]
       Company: [company]
       Location: [location]
       Job Description: [description]"
   max_tokens: 1000
   Returns: { text: coverLetterText, filePath: savedFilePath }

2. async saveCoverLetterDocx(job, coverLetterText)
   Creates the Word document as specified above
   Saves to data/cover-letters/ folder
   Returns: filePath of saved document

3. async generateForTierOneAndTwo(jobs)
   Input: array of job objects (all jobs)
   Filters to only tier === 1 or tier === 2
   For each qualifying job:
     Calls generateCoverLetter
     Logs: "Generating cover letter for [title] at [company]..."
     Adds 2 second delay between API calls to avoid rate limits
   Returns: array of { job, filePath, text } objects
   Logs summary: "Generated X cover letters in data/cover-letters/"

4. async generateSingleCoverLetter(job)
   Generates and saves ONE cover letter for a single job
   Returns: { text, filePath }
   This is for use from the dashboard (Session 20)
