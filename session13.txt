FUNCTIONS TO EXPORT:

1. isFortuneCompany(companyName)
   Returns true if company matches Fortune 500 list, false otherwise

2. isGCCConglomerate(companyName)
   Returns true if company matches GCC conglomerate list, false otherwise

3. scoreLocation(location)
   Returns points 5-25 based on location rules above

4. scoreCompany(companyName)
   Returns points 5-25 based on prestige rules above

5. scoreSalary(salaryText)
   Parses salary from text (handles AED, USD, GBP)
   Converts USD to AED using rate 1 USD = 3.67 AED
   Converts GBP to AED using rate 1 GBP = 4.75 AED
   Returns points 5-25 based on salary rules above

6. scoreProfileMatch(title, description)
   Matches against strong/medium/weak keywords above
   Returns points 5-25 based on match score

7. prioritizeJob(job)
   Takes a job object with fields: title, company, location, salary, description
   Calls all 4 scoring functions
   Returns the job object with these added fields:
     locationScore (0-25)
     companyScore (0-25)
     salaryScore (0-25)
     matchScore (0-25)
     totalScore (0-100)
     tier (1, 2, 3, or 4)
     tierLabel (APPLY TODAY / APPLY THIS WEEK / APPLY IF TIME / OPTIONAL)
     isFortuneCompany (true/false)
     isGCCConglomerate (true/false)
     fortuneBadge (string: "Fortune 500" or empty string)

8. prioritizeAllJobs(jobsArray)
   Runs prioritizeJob on every job in array
   Returns array sorted by totalScore descending
