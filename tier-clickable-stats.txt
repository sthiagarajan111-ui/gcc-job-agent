FIX 2 — CLICKABLE STATS BAR COUNTERS
═══════════════════════════════════════════════════════

In src/dashboard.js client-side JavaScript:

Make each stat box in the stats bar clickable.
When clicked, apply the relevant filter to show
only that category of jobs.

Add to each stat box:
  cursor: pointer
  hover effect: brightness(1.2) or subtle border glow

Click behavior for each stat box:

1. Total Jobs box clicked:
   → Reset ALL filters to default (show everything)
   → Same as clicking Reset Filters button

2. Today's New Jobs box clicked:
   → Set date filter to "Today" (filter-date = "today")
   → Apply filters
   → Scroll to job grid

3. Fortune 500 box clicked:
   → Toggle Fortune 500 filter ON
   → If already on → toggle OFF (reset to all)
   → Apply filters
   → Scroll to job grid

4. Tier 1 box clicked:
   → Set Tier filter to "Tier 1" (filter-tier = "1")
   → Apply filters
   → Scroll to job grid

5. Tier 2 box clicked:
   → Set Tier filter to "Tier 2" (filter-tier = "2")
   → Apply filters
   → Scroll to job grid

6. Applied box clicked:
   → This is a tracker stat — clicking it should
     navigate to the Tracker page:
     window.location.href = '/tracker'

Implementation:
Add onclick handlers to each stat box div:

stat-total onclick: resetFilters()
stat-today onclick: 
  document.getElementById('filter-date').value = 'today'
  applyFilters()
  document.getElementById('jobs-grid').scrollIntoView()
stat-fortune500 onclick:
  const btn = document.getElementById('filter-fortune500')
  btn.classList.toggle('active')
  applyFilters()
  document.getElementById('jobs-grid').scrollIntoView()
stat-tier1 onclick:
  document.getElementById('filter-tier').value = '1'
  applyFilters()
  document.getElementById('jobs-grid').scrollIntoView()
stat-tier2 onclick:
  document.getElementById('filter-tier').value = '2'
  applyFilters()
  document.getElementById('jobs-grid').scrollIntoView()
stat-applied onclick:
  window.location.href = '/tracker'

Add a small tooltip on each stat box:

  Fortune 500: "Click to filter Fortune 500 only"
  Tier 1: "Click to show Tier 1 jobs"
