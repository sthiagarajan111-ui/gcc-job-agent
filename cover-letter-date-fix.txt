FIX 2 — DATE OF POSTING FILTER + AUTO-DELETE OLD JOBS
═══════════════════════════════════════════════════════

PART A — Add Date Posted filter to filter bar:

Add a new dropdown filter after the Experience filter:

Label: "All Dates"
Options:
  All Dates
  Today
  Last 3 Days
  Last 7 Days
  Last 14 Days
  Last 30 Days
  Last 60 Days
  Last 90 Days

HTML id: "filter-date"
Style: same as other filter dropdowns

When selected, filter jobs to show only jobs where
the posting date is within the selected range.

Date field to check (in order of preference):
  job.postedDate || job.datePosted || job.date ||
  job.addedDate || job.createdAt

Date comparison logic:
  const today = new Date()
  const posted = new Date(job.postedDate || job.datePosted)
  const daysAgo = (today - posted) / (1000 * 60 * 60 * 24)

  Apply filter:
    Today: daysAgo <= 1
    Last 3 Days: daysAgo <= 3
    Last 7 Days: daysAgo <= 7
    Last 14 Days: daysAgo <= 14
    Last 30 Days: daysAgo <= 30
    Last 60 Days: daysAgo <= 60
    Last 90 Days: daysAgo <= 90

  If job has no date: always include it
  (don't hide jobs just because date is unknown)

Add filter-date to:
  - applyFilters() function
  - resetFilters() function (reset to "All Dates")

PART B — Auto-delete jobs older than 90 days:

In loadAllJobs() function, after loading all jobs
and before returning, filter out jobs older than
90 days:

const ninetyDaysAgo = new Date()
ninetyDaysAgo.setDate(ninetyDaysAgo.getDate() - 90)

Filter logic:
  For each job, get the posting date
  If posting date exists AND is older than 90 days:
    Add to blocked-jobs.json automatically
    Remove from the jobs array
  If no posting date: keep the job

Also add a console.log:
  console.log('Auto-removed X jobs older than 90 days')

This runs every time the dashboard starts — keeping
the job list clean automatically.

PART C — Show posting date age on job card:

The existing "Posted: X days ago" info row already
shows this. Make sure the color changes based on age:
  0-7 days: green #3FB950 (fresh)
  8-30 days: blue #58A6FF (recent)
  31-60 days: amber #D29922 (aging)
  61-90 days: orange #F78166 (old)
  Unknown: grey #6E7681
