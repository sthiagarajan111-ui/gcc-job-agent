═══════════════════════════════════════════════════════
FUNCTIONS TO EXPORT
═══════════════════════════════════════════════════════

1. async fillForm(url, job)
   Input:
     url: the job application page URL
     job: object with title, company, location fields
   Steps:
     a. Launch Puppeteer with headless: false so browser is visible
        Set defaultViewport: null for full window
        Set args: ['--start-maximized']
     b. Open the URL
     c. Wait for page to load: waitForSelector('form', timeout 15000)
        If no form found after 15s: log warning and continue anyway
     d. Call fillAllFields(page) to fill all detected fields
     e. Call injectOverlay(page, filledCount) to show the overlay
     f. Log: "Form auto-filled for [title] at [company]"
        Log: "Browser is open — please review and submit manually"
        Log: "Fields filled: [X]"
     g. Do NOT close the browser — leave it open for Dheeraj
     h. Return: { filledCount, browser, page }

2. async fillAllFields(page)
   Gets all input, textarea and select elements on the page
   For each element:
     Reads name, id, placeholder, aria-label, type attributes
     Tries to match against the field mapping above
     If match found:
       For input/textarea: use page.evaluate to set value directly
         AND dispatch an input event and change event so the
         form registers the value (important for React/Angular forms)
       For select dropdowns: try to select the best matching option
       Apply yellow highlight
       Increment filledCount
     Skips visa/work permit fields completely
   Returns: filledCount (number of fields filled)

3. async injectOverlay(page, filledCount)
   Injects the fixed overlay div as described above
   Replaces [X] in the text with the actual filledCount number

4. async autoFillJobApplication(job)
   Main entry point called from dashboard (Session 20)
   Input: job object with applyUrl field
   Calls fillForm(job.applyUrl, job)
   Returns result from fillForm
