═══════════════════════════════════════════════════════
FUNCTIONS TO EXPORT
═══════════════════════════════════════════════════════

1. estimateSalary(title, location)
   Looks up the market rate table for best match
   Matches on role keywords in title (case-insensitive)
   Matches on location keywords
   If exact match found: returns the table entry
   If no match: returns a default estimate based on location:
     Dubai default:       { minAED: 15000, maxAED: 25000, typicalAED: 20000 }
     Abu Dhabi default:   { minAED: 14000, maxAED: 23000, typicalAED: 18000 }
     Qatar default:       { minAED: 13000, maxAED: 22000, typicalAED: 17000 }
     Saudi Arabia default:{ minAED: 12000, maxAED: 20000, typicalAED: 16000 }
     Kuwait default:      { minAED: 11000, maxAED: 18000, typicalAED: 14000 }
     Bahrain default:     { minAED: 10000, maxAED: 16000, typicalAED: 13000 }
     Oman default:        { minAED: 9000,  maxAED: 14000, typicalAED: 11000 }
     Unknown default:     { minAED: 12000, maxAED: 20000, typicalAED: 16000 }
   Returns object with:
     minAED, maxAED, typicalAED,
     matchedRole (string or "default estimate"),
     confidence ("high" if table match, "estimated" if default)

2. getAutoFillSalary(title, location)
   Calls estimateSalary internally
   Logic:
     If typicalAED >= 18000: return 18000 (Dheeraj's target)
     If typicalAED >= 14000: return typicalAED
     If typicalAED < 14000:  return 14000 (never go below floor)
   Returns object with:
     autoFillAED (the value to type into salary fields)
     minimumAED: 14000
     targetAED: 18000
     stretchAED: 25000
     recommendation (string explaining the value)

3. convertToAED(amount, currency)
   Converts any currency amount to AED
   Supports: GBP, USD, EUR, SAR, QAR, KWD, AED
   Returns number rounded to nearest integer

4. parseSalaryFromText(text)
   Extracts salary numbers from job description text
   Handles formats like:
     "AED 18,000", "18000 AED", "AED18000"
     "USD 5,000", "$5000"
     "GBP 3,500", "£3500"
     "15,000 - 20,000 AED"
     "up to AED 25000"
   Returns object with:
     found (true/false)
     currency (AED/USD/GBP/unknown)
     minAED (converted to AED)
     maxAED (converted to AED)
     rawText (the matched portion)

5. getSalaryReport(title, location, salaryText)
   Master function combining all the above
   If salaryText is provided and parseable: use parsed value
   If salaryText empty or unparseable: use estimateSalary
   Returns full report object with:
     postedSalary (parsed or null)
     estimatedSalary (from table)
     autoFillRecommendation (from getAutoFillSalary)
     allowances (the full allowances reference)
     comparisonToUK: {
       ukMonthlyTakehomeGBP: 2380,
       ukMonthlyTakehomeAED: 14000,
       gccTypicalAED: (typicalAED value),
       realIncreasePercent: calculated percentage
     }
     verdict (string: one of these three)
       "ABOVE TARGET" if typicalAED >= 18000
       "AT MINIMUM"   if typicalAED >= 14000
       "BELOW FLOOR"  if typicalAED < 14000
