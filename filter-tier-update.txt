THE CHANGE — TIER-DEPENDENT COMPANY DROPDOWN
═══════════════════════════════════════════════════════

When the Tier filter changes, the Company dropdown
must immediately repopulate with ONLY the companies
that belong to the selected tier.

LOGIC:

When Tier = "All Tiers":
  Company dropdown shows ALL companies from all jobs
  sorted alphabetically

When Tier = "Tier 1 Only":
  Company dropdown shows ONLY companies where
  job.tier === 1
  sorted alphabetically

When Tier = "Tier 2 Only":
  Company dropdown shows ONLY companies where
  job.tier === 2
  sorted alphabetically

When Tier = "Tier 3 Only":
  Company dropdown shows ONLY companies where
  job.tier === 3
  sorted alphabetically

When Tier = "Tier 4 Only":
  Company dropdown shows ONLY companies where
  job.tier === 4
  sorted alphabetically

Always keep "All Companies" as the first option
after repopulating.

After repopulating the company dropdown:
  Reset its selected value to "All Companies"
  Then re-run filterJobs() to update the visible cards

WHERE TO ADD THIS LOGIC
═══════════════════════════════════════════════════════

Find the existing Tier dropdown change event listener.
It currently calls filterJobs() when tier changes.

Add this BEFORE the filterJobs() call:

function updateCompanyDropdown(allJobs, selectedTier) {
  // Filter jobs by tier
  let tierJobs = allJobs
  if (selectedTier === '1') tierJobs = allJobs.filter(j => j.tier === 1)
  else if (selectedTier === '2') tierJobs = allJobs.filter(j => j.tier === 2)
  else if (selectedTier === '3') tierJobs = allJobs.filter(j => j.tier === 3)
  else if (selectedTier === '4') tierJobs = allJobs.filter(j => j.tier === 4)

  // Get unique companies from filtered jobs
  const companies = ['All Companies',
    ...new Set(tierJobs.map(j => j.company)
    .filter(Boolean).sort())]

  // Repopulate company dropdown
  const companySelect = document.getElementById('filter-company')
  companySelect.innerHTML = companies.map(c =>
    `<option value="${c}">${c}</option>`).join('')

  // Reset to All Companies
  companySelect.value = 'All Companies'
}

Call updateCompanyDropdown(allJobs, selectedTierValue)
every time the Tier dropdown changes, passing the
full unfiltered allJobs array and the new tier value.

Note: allJobs must be stored as a variable accessible
to this function — it should already be available as
the full jobs array loaded from /api/jobs.
If not, store it in a variable called allJobs at the
top of the script when jobs are first loaded.

function updateSourceDropdown(allJobs, selectedTier) {
  let tierJobs = allJobs
  if (selectedTier === '1') tierJobs = allJobs.filter(j => j.tier === 1)
  else if (selectedTier === '2') tierJobs = allJobs.filter(j => j.tier === 2)
  else if (selectedTier === '3') tierJobs = allJobs.filter(j => j.tier === 3)
  else if (selectedTier === '4') tierJobs = allJobs.filter(j => j.tier === 4)

  const sources = ['All Sources',
    ...new Set(tierJobs.map(j =>
      j.source || j.platform || j.board || 'Unknown')
    .filter(Boolean).sort())]

  const sourceSelect = document.getElementById('filter-source')
  sourceSelect.innerHTML = sources.map(s =>
    `<option value="${s}">${s}</option>`).join('')

  sourceSelect.value = 'All Sources'
}

Call updateSourceDropdown(allJobs, selectedTierValue)
every time the Tier dropdown changes.
