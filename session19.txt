═══════════════════════════════════════════════════════
FUNCTIONS TO EXPORT
═══════════════════════════════════════════════════════

1. generateEmailHTML(jobs, pipelineSummary)
   Input:
     jobs: array of prioritized job objects from
           jobPrioritizer.js (with tier, scores, etc.)
     pipelineSummary: object from appTracker.js
   Builds and returns the complete HTML string
   Uses salaryEngine.js getSalaryReport for jobs
   with no posted salary
   All salary calculations done inline

2. async sendReportEmail(jobs, pipelineSummary)
   Calls generateEmailHTML to build the HTML
   Sends via nodemailer to all 4 addresses
   Saves HTML to: data/reportV2-YYYY-MM-DD.html
   Logs: "ReporterV2 email sent to 4 recipients"
   Returns: { success, filePath, tier1Count, tier2Count }

3. async runDailyReport(jobs)
   Master function called from index.js
   Loads pipeline summary from appTracker.js
   Calls sendReportEmail(jobs, pipelineSummary)
   Returns result
