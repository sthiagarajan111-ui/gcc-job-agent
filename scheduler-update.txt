Update the existing file: setup-task.ps1

Read the current setup-task.ps1 first before making changes.

Change the scheduled task trigger time from 8:00 AM Gulf time
to 8:10 AM Gulf time.

Gulf time is UTC+4.
8:10 AM Gulf time = 4:10 AM UTC.

So the scheduled task trigger time must be set to 04:10 UTC.

Find the current time setting in the file.
It will be something like:
  -At "04:00AM"
  OR a cron expression: 0 4 * * *
  OR a trigger time of 04:00

Change it to:
  -At "04:10AM"
  OR cron: 10 4 * * *
  OR trigger time 04:10

After updating setup-task.ps1, re-register the
Windows Task Scheduler task by running:

powershell -ExecutionPolicy Bypass -File setup-task.ps1

Then verify the task is registered with the new time:
  Get-ScheduledTask -TaskName "GCCJobAgent" | 
  Get-ScheduledTaskInfo

Or whatever the task name is in the file.

Print: "Task scheduler updated to 8:10 AM Gulf time (04:10 UTC)"
Print: "Task re-registered successfully"
Print: "Next run time: [next scheduled run time]"
